# FormCraft Pro

FormCraft Pro یک افزونه پیشرفته فرم‌ساز برای وردپرس است که شامل قابلیت‌های زیر می‌باشد:

- سازنده فرم با رابط کاربری React
- REST API برای مدیریت فرم‌ها
- پایگاه داده برای ذخیره فرم‌ها، فیلدها و ورودی‌ها
- رابط کاربری مدرن با Tailwind CSS

## نصب و راه‌اندازی

### پیش‌نیازها
- Node.js (نسخه 16 یا بالاتر)
- npm یا yarn
- وردپرس (نسخه 5.8 یا بالاتر)

### مراحل نصب

1. نصب وابستگی‌ها:
```bash
cd formcraft-pro
npm install
```

2. ساخت فایل‌های فرانت‌اند:
```bash
npm run build
```

3. کپی پوشه به وردپرس:
```bash
# پوشه formcraft-pro را به wp-content/plugins/ کپی کنید
```

4. فعال‌سازی پلاگین:
- به داشبورد وردپرس بروید
- به بخش Plugins بروید
- "FormCraft Pro" را فعال کنید

## توسعه

برای توسعه محلی با HMR:
```bash
npm run dev
```

## ساختار پروژه

```
formcraft-pro/
├─ assets/              # فایل‌های استاتیک
├─ build/               # خروجی build (تولید شده)
├─ src/
│  ├─ admin/            # کد React
│  └─ php/              # کنترلرها و سرویس‌های PHP
├─ includes/            # کلاس‌های اصلی پلاگین
├─ formcraft-pro.php    # فایل اصلی پلاگین
└─ package.json
```
