# راهنمای نصب FormCraft Pro

## مراحل نصب و راه‌اندازی

### ۱. آماده‌سازی فایل‌ها

پروژه الان کاملاً آماده است و فایل‌های build هم ساخته شده‌اند.

### ۲. انتقال به وردپرس

کل پوشهٔ `formcraft-pro` را به مسیر زیر در سرور وردپرس خود کپی کنید:

```
wp-content/plugins/formcraft-pro
```

### ۳. فعال‌سازی پلاگین

1. وارد داشبورد مدیریت وردپرس شوید
2. به منوی **افزونه‌ها** (Plugins) بروید
3. پلاگین **FormCraft Pro** را پیدا کنید
4. روی دکمهٔ **فعال‌سازی** (Activate) کلیک کنید

### ۴. استفاده از پلاگین

بعد از فعال‌سازی:
- در منوی سمت چپ داشبورد، آیتم جدیدی به نام **FormCraft** اضافه می‌شود
- روی آن کلیک کنید تا وارد صفحهٔ مدیریت فرم‌ها شوید
- می‌توانید فرم جدید ایجاد کنید

## توسعه و سفارشی‌سازی

اگر می‌خواهید روی کد کار کنید:

### نصب وابستگی‌ها
```bash
cd formcraft-pro
npm install
```

### اجرای حالت توسعه (Development)
```bash
npm run dev
```

### ساخت نسخهٔ نهایی
```bash
npm run build
```

## ساختار دیتابیس

پلاگین به صورت خودکار ۳ جدول در دیتابیس ایجاد می‌کند:

- `wp_fcp_forms` - ذخیرهٔ فرم‌ها
- `wp_fcp_fields` - ذخیرهٔ فیلدهای هر فرم
- `wp_fcp_entries` - ذخیرهٔ ورودی‌های ارسال شده

## REST API

پلاگین شامل REST API endpoints زیر است:

- `GET /wp-json/fcp/v1/forms` - دریافت لیست فرم‌ها
- `POST /wp-json/fcp/v1/forms` - ایجاد فرم جدید
- `GET /wp-json/fcp/v1/entries/{id}` - دریافت یک ورودی

## نکات امنیتی

- تمام endpointها نیاز به دسترسی `manage_options` دارند
- از nonce برای احراز هویت استفاده می‌شود
- تمام ورودی‌ها sanitize می‌شوند
- کوئری‌های دیتابیس از prepared statements استفاده می‌کنند

## پشتیبانی

در صورت بروز مشکل، لاگ‌های مرورگر (Console) و لاگ‌های وردپرس را بررسی کنید.
