🎉 راهنمای نصب پلاگین FormCraft Pro در وردپرس

📦 فایل آماده: formcraft-pro-wordpress.zip (67 KB)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 مراحل نصب:

1️⃣ ورود به داشبورد وردپرس
   - به آدرس: your-site.com/wp-admin بروید
   - با اطلاعات مدیریتی وارد شوید

2️⃣ رفتن به بخش افزونه‌ها
   - از منوی سمت چپ، روی "Plugins" کلیک کنید
   - روی دکمه "Add New" کلیک کنید

3️⃣ آپلود فایل زیپ
   - روی دکمه "Upload Plugin" کلیک کنید
   - روی "Choose File" کلیک کنید
   - فایل "formcraft-pro-wordpress.zip" را انتخاب کنید
   - روی "Install Now" کلیک کنید

4️⃣ فعال‌سازی پلاگین
   - بعد از نصب، روی "Activate Plugin" کلیک کنید
   - پلاگین فعال می‌شود و جداول دیتابیس ایجاد می‌شوند

5️⃣ استفاده از پلاگین
   - در منوی سمت چپ، آیتم جدید "FormCraft" اضافه شده است
   - روی آن کلیک کنید
   - می‌توانید فرم جدید ایجاد کنید

🔧 پیش‌نیازها:

✓ وردپرس نسخه 5.8 یا بالاتر
✓ PHP نسخه 7.4 یا بالاتر
✓ MySQL 5.6 یا بالاتر
✓ دسترسی مدیریتی (manage_options)

📊 جداول دیتابیس (خودکار ایجاد می‌شوند):

• wp_fcp_forms       - ذخیره فرم‌ها
• wp_fcp_fields      - ذخیره فیلدهای فرم
• wp_fcp_entries     - ذخیره ورودی‌های ارسال شده

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🌐 REST API Endpoints:

GET  /wp-json/fcp/v1/forms          - دریافت لیست فرم‌ها
POST /wp-json/fcp/v1/forms          - ایجاد فرم جدید
GET  /wp-json/fcp/v1/entries/{id}   - دریافت یک ورودی

❓ عیب‌یابی:

🔴 اگر پلاگین فعال نمی‌شود:
   - مجوزهای فایل‌ها را چک کنید (755 برای پوشه‌ها، 644 برای فایل‌ها)
   - لاگ‌های PHP را بررسی کنید

🔴 اگر صفحه خالی است:
   - Console مرورگر را باز کنید (F12)
   - مطمئن شوید فایل‌های build وجود دارند

🔴 اگر API کار نمی‌کند:
   - به Settings > Permalinks بروید
   - روی "Save Changes" کلیک کنید (بدون تغییر)
